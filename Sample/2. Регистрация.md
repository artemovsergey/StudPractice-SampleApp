# Регистрация

Создаём страницу  ```SignPage``` привязываем кнопку к странице
регистрации на главной странице.

```html
<a class="btn btn-lg btn-primary" href="#">Регистрация</a>
```



Оформляем полную структуру страницы ```SignPage``` и проставляем значение в
тег <title>. Ниже представлено начальное содержимое страницы ```SignPage```. 

```html
<h1>Sign up</h1>
<p>This will be a signup page for new users.</p>
```

# Создание формы регистрации

Откройте файл ```SignPage.cshtml```. Ниже представлена форма регистрации новых пользователей:

```html
<div class="row">
 <div class="col-md-6 col-md-offset-3">
 <form action="sign.php" method="POST" class="form-signin" role="form">
 <input type="text" name="name" class="form-control" placeholder="Name"
required autofocus>
 <input type="email" name="email" class="form-control" placeholder="Email"
required autofocus>
 <input type="password" name="password" class="form-control"
placeholder="Password" required autofocus>
 <input type="password" name="password_confirmation" class="form-control"
placeholder="Confirmation" required>
 <button class="btn btn-lg btn-primary btn-block" type="submit"
name="submit">Create user </button>
 </form>
 </div>
 </div>
```

![]()

Стили CSS для формы регистрации:

```css
input, textarea, select, .uneditable-input {
border: 1px solid #bbb;
width: 100%;
margin-bottom: 15px
@include box_sizing;
}
input {
height: auto !important;
}
```

# Проверка регистрации

Далее нам надо осуществить первичную регистрацию. После регистрации мы сразу же сделаем редирект на страницу ```IndexPage```. В результате в базу данных добавиться новый пользователь.
