# Регистрация

Создаём страницу регистрации ```SignPage``` привязываем кнопку к странице
регистрации на главной странице.

```html
<a class="btn btn-lg btn-primary" href="sign.php">Регистрация</a>
```
Оформляем полную структуру страницы sign.php и проставляем значение в
тег <title>. Ниже представлено начальное содержимое страницы sign.php. 

```html
<h1>Sign up</h1>
<p>This will be a signup page for new users.</p>
```

# Создание формы регистрации

Теперь, получив действующую (хоть и незавершенную) страницу
профиля пользователя, можно приступить к созданию формы для регистрации
на нашем сайте. Мы видели, что страница регистрации в настоящий момент
пуста и совершенно бесполезна. Цель этого раздела – начать менять это
печальное положение дел и создать форму.
Откройте файл sign.php. Ниже представлена форма регистрации новых
пользователей 

```html
<div class="row">
 <div class="col-md-6 col-md-offset-3">
 <form action="sign.php" method="POST" class="form-signin" role="form">
 <input type="text" name="name" class="form-control" placeholder="Name"
required autofocus>
 <input type="email" name="email" class="form-control" placeholder="Email"
required autofocus>
 <input type="password" name="password" class="form-control"
placeholder="Password" required autofocus>
 <input type="password" name="password_confirmation" class="form-control"
placeholder="Confirmation" required>
 <button class="btn btn-lg btn-primary btn-block" type="submit"
name="submit">Create user </button>
 </form>
 </div>
 </div>
```

![]()

Стили CSS для формы регистрации 

```css
input, textarea, select, .uneditable-input {
border: 1px solid #bbb;
width: 100%;
margin-bottom: 15px
@include box_sizing;
}
input {
height: auto !important;
}
```
# Проверка регистрации

Подключим базу данных в файле sign.php. Далее нам надо осуществить
первичную регистрацию. После регистрации мы сразу же сделаем редирект на
страницу home.php. В результате в базу данных добавиться новый
пользователь.
